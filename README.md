# Explanation: 
Prompt:
"Design a personal productivity assistant that can help manage daily tasks, schedule reminders, suggest wellness tips, and answer general queries. The assistant should interact using natural language and be adaptable to the user’s changing preferences over time."
Procedure:
1. Define the core requirements of a personal productivity assistant.
2. Identify and construct appropriate prompts for each task using an LLM (e.g., ChatGPT).
3. Simulate natural user interaction through a simple interface or command-line system.
4. Collect feedback or inputs from users and adapt responses accordingly.
5. (Optional) Integrate basic memory to simulate preference adaptation.
EXPECTED OUTPUT: - (attached the drive link)
Output (Example Response by LLM):
Personal Productivity Assistant Features:
1. Daily Task Manager:
o Accept tasks via natural language (e.g., "Remind me to call mom at 6 PM").
o Organize tasks by priority and deadline.
o Provide daily summaries and pending items.
2. Smart Scheduler:
o Schedule events and set reminders using contextual understanding.
o Notify user of overlapping appointments or free time slots.
3. Wellness Tips Generator:
o Suggest daily wellness advice (hydration, exercise, screen-time breaks).
o Adapt suggestions based on past user preferences and responses.

# Expected Output:
**Application Name: Daily Task Organizer using ChatGPT**

**Functionality:**
The application allows the user to:

Add, remove, and view daily tasks.

Categorize tasks by priority or type.

Get suggestions for task optimization and reminders.

Generate a daily summary with actionable insights.

**1. Simple Prompt (Basic Task Addition)**
**User Input:** “Add ‘Finish assignment’ to my tasks for today.”
**ChatGPT Output:**

Task “Finish assignment” has been added to today’s list.

You have 1 task today.

**2. Zero-Shot Prompt (Direct Instruction without Examples)**
**User Input:** “Organize my tasks for today and prioritize urgent ones.”
**ChatGPT Output:**

**Urgent Tasks:**

Submit project report

Call the doctor

**Regular Tasks:**

Water plants

Read a book

**3. Few-Shot Prompt (Providing Examples to Guide AI)**
**Prompt Example:**

**Example 1:** “Add ‘Buy groceries’ to my shopping list.” → “Task ‘Buy groceries’ added to shopping list.”

**Example 2:** “Mark ‘Submit assignment’ as completed.” → “Task ‘Submit assignment’ marked as completed.”
User Input: “Add ‘Prepare presentation’ to work tasks.”
**ChatGPT Output:**

Task “Prepare presentation” added to work tasks.

You now have 3 tasks under Work category.

**4. Chain-of-Thought Prompt (Step-by-Step Reasoning)**
User Input: “Plan my day efficiently.”
ChatGPT Output:

**Analyze tasks for today:**

**Urgent:** Submit project report, Call doctor

**Regular:** Water plants, Read book

**Allocate time blocks:**

9:00–11:00: Submit project report

11:00–11:30: Call doctor

4:00–4:30: Water plants

5:00–6:00: Read book

**Suggest task optimization:** Combine similar tasks to save time.

**Daily Summary Generated:**

Total Tasks: 4

Urgent Completed: 0

**Expected Demonstration Output Table:**

Recommended Priority Order: Submit project report → Call doctor → Water plants → Read book

| Prompt Type      | Input Example               | Output Example                                                            |
| ---------------- | --------------------------- | ------------------------------------------------------------------------- |
| Basic Prompt     | Add ‘Finish assignment’     | Task added. You have 1 task today.                                        |
| Zero-Shot        | Organize my tasks for today | Tasks categorized into urgent and regular.                                |
